# Insta-Profile-osint
# 📸 Instagram Profile Analyzer

A powerful Python-based Instagram Profile Analyzer built using the `requests` module. This tool fetches and analyzes publicly available Instagram profile information and presents useful insights in a clean and structured format.

---

## 🚀 Features

* 🔍 Analyze public Instagram profiles
* 👤 Fetch profile information
* 📊 Extract profile statistics
* 🖼️ Get profile picture URL
* 👥 Followers & Following count
* 📝 Bio and account details
* ⚡ Fast and lightweight
* 🐍 Built with pure Python and Requests

---

## 📦 Requirements

* Python 3.8+
* Requests

Install dependencies:

```bash
pip install requests
```

---

## ⚙️ Installation

Clone the repository:
